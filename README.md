# pavani8787.github.io
A creative and interactive birthday greeting web page featuring animated card transitions, candle blow effect, confetti celebration, and a surprise message reveal. Built using pure HTML, CSS, and JavaScript.
# 🎉 Happy Birthday Card for Charitha! 🎂💖

A fun and interactive **HTML birthday card** that lets you blow the candle, open the gift, and reveal a secret message with animations and confetti!  

---

## 🌈 Features

- 🎂 **Blow the Candle** – Click to make the candle flame disappear and trigger confetti.  
- 🎁 **Open the Gift** – Card flips to reveal a birthday message.  
- ✨ **Secret Surprise** – A hidden panel appears with a special message for Charitha.  
- 🌸 **Floating Decorations** – Emojis float up in the background for a lively effect.  
- 🌟 **Smooth Animations** – Sliding card, flickering flame, and gradient background.  

---

## 🛠 How to Use

1. Clone or download this project.  
2. Open the `index.html` file in any modern browser.  
3. Click the card to interact:  
   - Step 1: Blow the candle 🕯️  
   - Step 2: Open your gift 🎁  
   - Step 3: Reveal the secret 💖  

---

## 💻 Technologies Used

- HTML5  
- CSS3 (animations, gradients, transitions)  
- JavaScript (DOM manipulation & confetti)  
- Google Fonts: `Pacifico` & `Poppins`  
- [canvas-confetti](https://www.npmjs.com/package/canvas-confetti) 🎊  

---

## 🎨 Customization

- Change the **recipient name** in the `<h1>` tag.  
- Customize the **colors** in the `:root` CSS variables.  
- Add your own **floating emojis** in the `icons` array inside the `createFloatingDecor` function.
